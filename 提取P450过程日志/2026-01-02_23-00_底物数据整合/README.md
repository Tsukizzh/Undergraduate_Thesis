## 关于22个P450无底物数据

这22个P450在enzymes.csv中有序列，但在data.csv中没有配对数据。这是正常现象，因为：
- enzymes.csv是**酶注册表**（包含所有酶的序列）
- data.csv是**训练数据**（只包含有实验配对数据的条目）
- 不是所有注册的酶都有用于ML训练的底物配对数据
