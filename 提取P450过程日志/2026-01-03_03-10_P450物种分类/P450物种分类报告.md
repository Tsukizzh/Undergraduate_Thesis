## 分类方法论

### 设计原则（Codex + Gemini协作）

1. **Codex建议**：
   - 使用UniProt REST API获取 `organism_name`, `organism_id`, `lineage`
   - 先保留个体物种，再按lineage分组
   - 使用Superkingdom/Domain + Eukaryote细分

2. **Gemini建议**：
   - 功能导向的混合分类法
   - 特别标记BIA生产植物（毛茛目）
   - CYP80和CYP719主要存在于毛茛目植物

### 分类逻辑

```python
if "bacteria" in lineage:
    category = "Bacteria"
elif "archaea" in lineage:
    category = "Archaea"
elif "fungi" in lineage:
    category = "Fungi"
elif "viridiplantae" in lineage:
    if any(order in lineage for order in ["Ranunculales"]):
        category = "Plants_BIA"
    else:
        category = "Plants_Other"
elif "metazoa" in lineage:
    if "mammalia" in lineage:
        category = "Mammals"
    else:
        category = "Other_Animals"
else:
    category = "Unknown"
```
